Acerca de Te Pillé

Te Pillé es una extensión de navegador avanzada diseñada para detectar sitios web construidos con el framework Lovable.

Características

  • 11 métodos de detección diferentes para máxima precisión
  • Soporte completo para SPAs (Single Page Applications)
  • Detección en tiempo real de contenido dinámico
  • Historial de las últimas 100 detecciones
  • Estadísticas de uso y contadores

Tecnología

Utiliza análisis avanzado del DOM, interceptación de History API, MutationObserver y múltiples estrategias de búsqueda para identificar Lovable en cualquier escenario.
